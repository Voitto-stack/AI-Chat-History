用户有每日美股分析任务，通过 Chrome MCP 自动化访问 https://www.investing.com/ 获取和分析美股数据。

## 任务描述
- **目标网站**：https://www.investing.com/
- **工具**：Chrome MCP（浏览器自动化）
- **频率**：每天执行

## Why: 动机
用户需要每天从 investing.com 获取美股数据进行分析，使用 Chrome MCP 可以自动化这个过程。

## How to apply: 执行方式
当用户提到"美股分析"时，应该：
1. 使用 Chrome MCP 工具打开 investing.com
2. 提取相关美股数据
3. 进行分析并输出结果

需要明确的细节：
- 具体分析哪些股票或指数？
- 需要提取哪些数据（价格、涨跌幅、技术指标等）？
- 分析结果如何输出（飞书推送 or 本地报告）？
- 是否需要自动化定时执行？
