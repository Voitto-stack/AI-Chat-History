## 可用能力

**已安装的工具**（位于 `~/.claude/tools/`）：

1. **创建文档**: `feishu-create-doc.js`
   - 支持从 Markdown 文件或直接内容创建
   - 自动转换格式并设置权限
   - 返回文档 URL

2. **发送消息**: `feishu-send.js`
   - 发送卡片消息到飞书群
   - 支持彩色标签

3. **读取文档**: `fd` CLI
   - 读取已有的飞书文档/wiki 内容

## 典型使用场景

```bash
# 1. 创建文档（从 Markdown 文件）
DOC_URL=$(feishu-create-doc.js "文档标题" /path/to/file.md | grep "https://" | tail -1)

# 2. 发送到飞书群
feishu-send.js "📄 新文档" "查看：$DOC_URL" green
```
