# Minerva

Minerva 后台管理系统前端，基于 Umi Max + Ant Design 构建。

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建
pnpm build
```

访问 http://localhost:8000

## 技术栈

- Umi Max 4.x
- Ant Design 5.x
- TypeScript
- React 19
